Barnsley's Fern,

also knows as Barnsley's Fractal. We chose this mathematical shape to be the symbol

and the code-artist project of our team, as it resembles perfectly the name of our team.

Because the name of our team is "Last but not Least", and the idea of a fractal is that

it is an infinite shape, the "Last" point drawn on the canvas cannot be the "Least", as

there can always be one more leaflet of the fern drawn after this "Least" point.


How does the fern work?

Barnsley's Fern is a fractal which resembles a fern, where each leaflet is a smaller

version of the entire fern, and thus this shape can be drawn to infinity. This fractal

was described in Michael Barnsley's book "Fractals Everywhere". Thorough mathematical

explanation/formula of it being formed can be found in the scripts.js file in the repository

of this project or here.
